# Pixel-Dash

## Project Overview

Pixel Dash is a 2D endless runner game developed for the Software Engineering course.

The player must avoid obstacles and survive as long as possible to achieve a high score.

## Functions

* Jump
* Obstacle Generation
* Collision Detection
* Score System
* Restart Function
