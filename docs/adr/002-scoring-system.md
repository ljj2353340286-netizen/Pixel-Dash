# ADR 002: 점수 시스템 설계

## 상태 (Status)
제안됨 (Proposed)

## 배경 (Context)
게임 내에서 플레이어의 실력을 측정할 수 있는 메커니즘이 필요합니다. 'Pixel-Dash'와 같은 러닝 게임에서 점수를 산정할 때 '생존 시간'을 기준으로 할지, '아이템 획득'을 기준으로 할지 결정해야 합니다.

## 결정 (Decision)
복합적인 점수 시스템을 도입하기로 결정했습니다:
1. **시간 점수**: 플레이어가 생존하는 매 초당 10점을 부여합니다.
2. **아이템 점수**: 특정 아이템(예: 코인)을 획득할 때마다 50점을 추가로 부여합니다.

## 결과 (Consequences)
- **장점**: 플레이어에게 '장애물 회피'와 '아이템 수집'이라는 두 가지 동기를 동시에 부여하여 게임성을 높일 수 있습니다.
- **단점**: UI 화면에서 두 가지 변수가 실시간으로 합산되는 과정을 처리해야 합니다.
