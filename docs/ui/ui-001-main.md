# UI Design Document

## Website Pages

The project contains four required pages:

* `main.html` - project introduction and links
* `team.html` - team roles and responsibilities
* `game.html` - playable Pixel Dash game
* `form.html` - feedback form

## Game Screen

The game page contains the following elements:

* Canvas game area
* Player character
* Obstacles
* Score display
* Best score display
* Game Over message
* Restart button

## Layout

* Navigation is displayed at the top of every page.
* Score and best score are displayed above the game area.
* Player starts near the left side of the canvas.
* Obstacles move from right to left.
* Restart is available after Game Over.

## Design Goal

The interface should be simple, readable, and clean enough for a final presentation.
The design supports the software engineering process instead of focusing on complex visual effects.
