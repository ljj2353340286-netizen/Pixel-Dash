# Test 006: Feedback Form

## Objective

Verify that the project feedback form works with JavaScript submit handling.

## Preconditions

Open `form.html` in a web browser.

## Steps

1. Enter a name.
2. Select a role.
3. Enter feedback text.
4. Click Submit Feedback.

## Expected Result

The form displays a confirmation message and clears the input fields.

## Status

Ready for manual testing.
